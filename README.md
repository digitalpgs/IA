# Digital Pages AI Showcase

This project is a professional single-page website about artificial intelligence, the best tools available, practical use cases, implementation steps, and pricing information.

## Features
- Responsive modern design
- Spanish, English and Catalan language switcher
- Clear sections for AI overview, tool comparison, process, and pricing
- Ready to publish via GitHub Pages

## Local preview
Open the project folder in a browser, or run:

```bash
python -m http.server 8000
```

Then visit http://127.0.0.1:8000/

## GitHub Pages
1. Push this folder to a GitHub repository.
2. In GitHub, open Settings > Pages.
3. Select the main branch and the root folder.
4. Publish the site.
